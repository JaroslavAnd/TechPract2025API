# TechPract2025API

## Опис
Простий REST API для керування подорожами.

## Запуск
1. Клонуйте репозиторій.
2. Відкрийте у Visual Studio.
3. Запустіть API (`Ctrl + F5` або `dotnet run`).
4. API доступне за `https://localhost:7242/swagger`.
